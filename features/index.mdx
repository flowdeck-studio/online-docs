## Core Development Features

### Building & Running
<Info>
  FlowDeck provides seamless integration with Xcode's build system, supporting both `.xcworkspace` and `.xcodeproj` files.
</Info>

Build your Swift projects with:
- **Smart caching** - Incremental builds that only recompile changed files
- **Parallel compilation** - Utilize all CPU cores for faster builds
- **Build configurations** - Support for Debug, Release, and custom configurations
- **Multiple schemes** - Switch between app targets, test targets, and more

[Learn more about building →](/features/build)

Deploy and test on real devices or simulators with ease:

## Performance Features

<Tip>
  Enable the Build Optimizer to reduce build times by up to 30%. [Learn more →](/performance/build-optimizer)
</Tip>

FlowDeck includes several performance optimizations:
- **Intelligent caching** - Reuse previously compiled modules
- **Parallel builds** - Compile multiple files simultaneously
- **Build optimizer** - Block unnecessary network calls during builds
- **Derived data management** - Automatic cleanup of build artifacts
