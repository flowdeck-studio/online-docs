---
title: "Swift Package Management"
description: "Manage your project's Swift Packages"
---

FlowDeck includes a Packages Dependencies panel that lists your project's swift packages an allows performing the most common operations.

You can access it via FlowDeck's panel \> Package Dependencies or run the commands directly from the Command Pallete.

## Available Commands

- **Update** **Packages **- Downloads and updates packages to the latest version
- **Resolve Package Versions** - Refresh the local copy of the selected package versions
- **Reset Package Caches** - Deletes SPM package caches and downloads/installs all project packages from scratch.
